# 第一章 消费者行为的影响因素理论

#### 1. 影响因素论的架构

* **架构类型**：两因素论；三因素论；四因素论。
* **消费者行为的三大影响因素**：
  1. 外部环境因素；
  2. 消费者内在因素；
  3. 市场营销因素。
* **消费者行为影响因素的四个层面**：
  * **文化因素**：文化，亚文化，社会阶层；
  * **社会因素**：参照群体，家庭角色与地位；
  * **个人因素**：个性和自我概念，年龄和生命周期阶段，职业经济环境，生活方式；
  * **心理因素**：激励，知觉，学习，信念和态度。

#### 2. 影响因素的清单

* **外部环境因素**：
  1. 文化和亚文化（价值观、传统、宗教、民族）；
  2. 社会消费基础结构（政策、消费基础设施、技术）；
  3. 家庭（结构、生命周期、决策模式）；
  4. 参照群体（资格、接触类型和吸引力）。
* **消费者内部因素**：
  5.  社会阶层（年龄、地位、收入、职业、教育）；
  6.  知觉、情绪、动机、态度、学习；
  7.  个性和自我概念；
  8.  世代和生活形态。
* **营销因素**：
  9.  营销传播（广告、促销、公关、消费者教育）；
  10. 营销要素（品牌、品质、服务、情境）。

#### 3. 消费者行为模型

* **消费者行为综合模型 (CTM)**：霍金斯模型 (Hawkins Model)。
* **消费者决策过程模型 (CDP)**：
  * **5阶段模型**：认知问题 → 搜寻信息 → 评价备选技术方案 → 选择与决策 → 购后评价。
  * **7阶段模型**：确认需求 → 搜集资料 → 购买前评估 → 购买 → 使用 → 用后评估 → 处理。

# 第二章 消费者需求、动机与价值理论

#### 1. 需要、动机和行为的关系

* **需要**：个体由于缺乏某种东西而产生的生理或心理上的不平衡的状态。它是消费行为的基础，没有需要就不会产生相应的消费行为。
* **动机**：被视为是一种个人内在的驱力，这种驱力促使个人采取行动。驱力主要来自因需要未得到满足而产生的紧张。
  * **类型**：积极/消极动机；理性/情感动机。
* **过程模型**：
  * 未被满足的需要/要求和欲望得到满足 → 紧张 → **驱力**（认知加工/紧张缓解） → **行为** → 目标实现或需要。

#### 2. 马斯洛需求层次论应用于购买动机研究

马斯洛理论把需求分成生理、安全、社交、尊重和自我实现五类。根据五个需求层次，可以划分出五个消费者市场（以洗衣粉为例）：

1. **生理需求**（满足最低需求层次）：只要求产品具有一般功能。
   * *洗衣粉案例*：关注“确实是洗衣粉”，选择价格最便宜的。
2. **安全需求**（满足对“安全”有要求）：关注产品对身体的影响。
   * *洗衣粉案例*：关注“洗衣粉质量好”，选择质量较好的。
3. **社交需求**（满足对“交际”有要求）：关注产品是否有助提高交际形象。
   * *洗衣粉案例*：关注“对交际的影响”（如精美包装、加香、柔顺），愿意付出更高价格。
4. **尊重需求**（满足与众不同要求）：关注产品的象征意义。
   * *洗衣粉案例*：关注“获得别人认可”（身份标志、优秀技术、特殊桶装），接受最高价格。
5. **自我实现**（满足自己判断规范）：拥有自己固定的品牌。
   * *洗衣粉案例*：对洗衣粉的认识转变为品牌对其生活的影响，在精神上认可某个品牌。

#### 3. 默里的心理需要清单（理解）

* 避免羞辱/防卫；施虐受虐；社会沟通；对无生命物的需要（获得、保护、秩序等）；反映抱负/权力/成就；人际间感情；与权力有关（支配、遵从等）。

#### 4. 霍金斯模型 (Hawkins Model)

* **外部影响**（文化、亚文化、体验与产品获取、人口环境、家庭、社会地位、参照群体、营销活动）
* **内部影响**（知觉、学习、记忆、动机、个性、情绪、态度）
  * 两者共同作用于 → **自我概念与生活方式** → **需要与欲望** → **决策过程**（问题识别、信息搜集、评价与选择、经销商选择和购买、购后过程）。

# 第三章 自我概念与生活形态理论

#### 1. 自我概念

* **定义**：对自我的看法，或对“我是谁”的理解。
* **原则**：自我的一致性、自我尊重的强化。

#### 2. 自我概念与营销行为

* **自我概念影响消费行为的两种动机**：

  1. **自我提升动机**：倾向于那些可以提升自我概念或者自我形象的行为。
  2. **自我一致性动机**：倾向于那些与自我概念相一致的行为。

  * *应用*：消费者在选购商品时，不仅依据质量价格，还依据**品牌个性是否符合自我概念**。

#### 3. 生活形态 (Lifestyle/生活方式)

* **定义**：人们花费时间和金钱的类型，反映了一个人的活动、兴趣和意见。
* **基本理论**：1. 价值观理论；2. 后结构理论。
* **营销意义**：描述目标市场；创造出关于市场的新看法；对产品进行定位；更好地传播产品特征；开发整合营销传播策略。

# 第四章 消费者信息处理及决策理论

#### 1. 消费者购买决策

* 指消费者谨慎地评价某一产品、品牌或服务的属性并进行理性选择、购买能满足某一特定需要的产品的过程。

#### 2. 消费者介入理论 (Involvement)

* **定义**：一个人基于内在需要、价值观和兴趣而感知到的与客体的关联性。
* **高介入**：主观上对商品感受越深（关联性强）。
* **低介入**：反之。

#### 3. 消费者购买决策过程的主要步骤

1. **确认需求**：来源（商品不足、新的信息、需求扩大）。
2. **寻求信息**：来源（个人经验、相关群众、大众媒体）。
3. **估价比较、决定购买**。
4. **购后评价**：预期满意理论；认识差距理论。

#### 4. 决策基本原则

* 最大满意原则；相对满意原则；遗憾最小原则；预期—满意原则。

#### 5. 态度 (Attitude)

* **定义**：以一种一贯的喜爱或不喜爱的方式对一个事物发生反应的习惯倾向。
* **ABC要素模型**：认知 (Cognition)、情感 (Affect)、意动 (Behavior)。

# 第六章 中国消费者行为分析导论

#### 1. 中国消费环境与西方不同

* **主要反映在**：消费政策；消费基础设施；变化的速度；消费者权益；消费能力等。
* **具体差异**：
  * 对未来的防范与保守花钱（储蓄率高）；
  * 购买决策（信息及时间跨度）比西方人慎重得多；
  * 产品的象征意义不同；
  * 价格敏感度强（对SP反应强）；
  * 购买者与使用者分离。
* **购买环境比较**：
  * **中国（高障碍）**：安全感低（假货、假广告），进入障碍高（入网费等），靠逼近消费者（广告轰炸）。
  * **美国（高吸力）**：放心度高（无因退货），诚信吸引。

#### 2. 结构性差异

* **时空两条线**：
  * **空间上**：区域消费差异大，“中国市场更似欧洲而非美国”（即横向差别大）。
  * **时间上**：世代差异十分明显（即纵向差别大）。
* **中国独生代**：1979年后出生，特殊的“一对夫妻只生一个孩子”群体，正在成为主力消费群体。

#### 3. 文化与消费者行为

* **文化的特性**：习得性、动态性、群体性、社会性、无形性。
* **文化价值观**：
  * **核心价值观**：居主导、核心地位（如成家立业）。
  * **次要价值观**：居从属、次要地位（如多子多福）。
  * **分类**：社会成员间关系；环境导向；自我导向。

#### 4. 中国文化的特色

* (一) 强大生命力和凝聚力；
* (二) 多样性与异质性；
* (三) 安土乐天的文化心态；
* (四) 人本主义；
* (五) 尊老崇古；
* (六) 重整体、倡协同。

#### 5. 中国文化的核心价值观

* **人道主义**：“仁”、“兼爱”、“仁爱孝悌”、“遵长爱幼”。
* **重义轻利**：“先义后利，以义制利”、“杀身成仁”、“舍生取义”。
* **理性优先**：“君子谋道不谋食”、“君子忧道不忧贫”。
* **诚信知报**：“投之木瓜，报之桃李”、“滴水之恩，当涌泉相报”。
* **贵和尚中**：“礼之用，和为贵”、“天时不如地利，地利不如人和”。
* **修己内圣**：通过修己、修身以达到“内圣”。
* **自强不息**：“人穷志不短”。
* **求是务实**：“知之为知之，不知为不知”。

#### 6. 中国的特色消费行为

* **面子文化（面子消费）**：涉及人群广泛，重“看”不重“用”，关注包装与文化寓意，地位决定档次。
  * *营销策略*：开辟送礼市场，个人化突显尊贵，豪华包装，节日投放广告。
* **根文化（根消费）**：崇拜消费又称神圣消费。包括祭祖、仪式、节庆、礼俗、房地产（落叶归根）。
* **关系文化（关系消费）**：礼尚往来，来而不往非礼也。

#### 7. 世代与群体分析

* **中国E世代**：
  * 特征：成长于网络时代，宽容、好奇心强、自我主张强、早熟、互动。
  * 消费：自主选择，量身订做，品牌忠诚度下降，注重功能而非形式。
* **中国独生代**：
  * 无所不闻超早熟，独立个性酷自我，全方位享乐主义，“有钱就花”不存钱（月光族），崇尚品牌时尚，旅游网游追求体验。
* **台湾E世代生活形态**：喜欢休闲、自我学习、PC/KTV、关心教育/社会问题、喜欢“加量不加价”。

# 区域消费差异 (TOFA 模型)

#### 1. 维度定义

* **时尚指数 S (Style)**：衡量在时尚——传统之间的区域位置。
* **花钱指数 R (Risk)**：衡量在勤俭——享受之间的区域位置。

#### 2. 四种基本类型

1. **A型（高S 高R）**：**前卫型**（Advance）。
   * 特征：时尚而敢花钱。追逐新潮，变化快，乐于接受外部文化，崇尚品牌，受群体影响大。
2. **F型（高S 低R）**：**理财型**（Fashion, Financing）。
   * 特征：时尚而精明。
   * *注：PDF原文此处可能有错位，通常指追求时尚但计算性价比。*
3. **O型（低S 高R）**：**乐天型**（Optimism）。
   * 特征：传统而敢花钱。决策快，敢冒消费风险，勇于尝试新品，主张享乐主义。
4. **T型（低S 低R）**：**保守型**（Traditionalism）。
   * 特征：传统而节俭。对外来文化谨慎抗拒，崇尚经典与传统价值，注重长远功能，对价格敏感。

# 中国文化与关系营销

#### 1. 概念

* **关系营销**：旨在建立、发展和维持关系交换的营销活动。
* 中国的人际关系有着丰富的文化内涵，体现着中国人的生活本身。

#### 2. 人际关系基本模式（四种关系）

1. **传承的关系**：
   * 与生俱来的（如血缘、亲缘、父母的老关系）。
   * **非工具性**，不是因为“有用”才建立。需要付出一定努力维持。
2. **嵌入的关系**：
   * 由于扮演各种社会角色而不得不发生的关系（如同事、同学）。
   * **非工具性**，受自己生活空间限制，在一定程度上可被控制。
3. **渐生的关系**：
   * 在生活过程中逐渐生成的关系。
   * **非工具性**，很大程度上受自己生活圈子限制，与嵌入关系重叠。
4. **开发的关系**：
   * 为了达到某种目的（一般是获取利益）而刻意发展的关系（如贸易伙伴）。
   * **工具性**，目的性非常强，需要计算利益得失。
