# 消费者行为学复习重点 (完整版)

### 第一篇：导论

#### 第一章：消费者行为研究概论

* **P2 消费者的概念**
  * **广义**：指购买、使用各种产品与服务的个人与组织。不仅包括个人、家庭，也包括企业、政府、其它社会组织。
  * **狭义**：指购买、使用各种消费品或服务的个人与家庭。
  * **本课程对象**：狭义的消费者。
* **P4 消费者行为概念**
  * 消费者行为是指消费者为获取、使用、处理消费物品所采用的各种行动以及事先决定这些行动的决策过程。
  * *(理解：5W1H - What, Why, Who, When, Where, How, How much)*
* **P12 炫耀性商品特点及例子**
  * **特点**：
    1. 既能带来物质效用又能带来虚荣效用；
    2. 源于对被人炫耀自己的社会心理而所能寄托的商品；
    3. 价格变动通常很大，波动容易被发现。
  * **例子**：20世纪初，纽约长岛和缅因州巴港的度假别墅。
* **P11 消费者行为研究的阶段**
  1. **萌芽时期**：研究范围狭窄，理论简单移植，未广泛重视。
  2. **应用时期**：专门研究展开，动机研究纷纷出现。
  3. **变革与发展时期**：加速发展，文献数量质量提高，成为独立学科，跨国界研究。
* **P4 购买角色**
  * **倡议者**：首先提出有意购买的人。
  * **影响者**：其看法建议有影响的人。
  * **决策者**：做出部分或全部决定的人。
  * **购买者**：实际购买的人。
  * **使用者**：实际消费或使用的人。

### 第二篇：消费者决策过程

#### 第二章：问题认知与信息搜索

* **P34 扩展型、有限型和名义型决策的区别**
  1. **介入程度**：名义型最低，复杂型（扩展型）最高，有限型居中。
  2. **重复选择概率**：复杂型重复选择同一品牌概率小，名义型重复选择可能性大。
  3. **信息搜寻时间**：名义型很少搜寻，复杂型广泛搜寻。
* **P37 促使解决问题的因素**
  1. 时间。
  2. 环境的改变。
  3. 产品获取。
  4. 产品消费。
  5. 个体差异。
* **P40 一般性问题认知 vs 选择型问题认知**
  * **区别**：
    * **一般性**：涉及理想与现实的差别，通过同类产品不同品牌缩小。扩大整体市场。
    * **选择性**：涉及差别通常只有特定品牌能解决。增加特定品牌销售。
  * **何时激发一般性**：
    1. 产品处于生命周期前期；
    2. 公司市场份额高；
    3. 问题认知后外部搜索有限；
    4. 需要全行业协作。
* **P42 信息获取来源**
  1. 记忆来源（内部）。
  2. 个人来源（亲友）。
  3. 大众/独立来源（媒体、消协）。
  4. 商业/营销来源（广告、推销员）。
  5. 经验来源（试用）。
* **P51 影响外部信息搜索的因素**
  * 购买活动重要性、备选品数量、同类店铺多少、可支配时间、身心状态、态度、知识多少、介入程度等。

#### 第三章：评价和购买

* **P54 评价标准**：消费者在选择备选品时考虑的产品属性或特征。
* **P64 冲动性购买**：无计划购买，进店前无意图，基于一时念头实施。
* **P74 购物导向**：强调某些活动的购物风格，与生活方式相关。
* **P64 营销消费者购买的三类因素**
  1. 他人的态度。
  2. 购买风险。
  3. 意外情况（收入变化、新产品出现等）。
* **P65 无计划购买对零售商启示**
  1. 注意力放在店内展示和促销；
  2. 研究无计划购买的程度与类型。
* **P74 购买的知觉风险与商店选择**
  1. 新兴/非传统渠道销售高风险产品，应设法降低风险。
  2. 折扣店应从有品牌影响的供应商进货。
  3. 传统零售店侧重销售高风险产品。
  4. 提供担保或保证。

#### 第四章：购后行为

* **P86 消费者满意**：对付出是否得到足够回报的认知状态，期望与实际的比较。
* **P89 消费者归因**：对他人或自己行为原因的推理过程。
* **P94 品牌忠诚**：对某品牌形成偏好、试图重复选择的倾向。
* **P87 影响满意的因素**
  1. 产品因素。
  2. 促销因素。
  3. 竞争品牌的影响。
  4. 消费者特征。
* **P94 品牌忠诚的重要性**
  1. 减轻竞争压力。
  2. 减少额外信息搜索，易受促销吸引。
  3. 价格敏感度低。
  4. 正面的口传。
* **P95 形成品牌忠诚的原因**
  1. 产品吸引。
  2. 时间压力（忠诚可缓解压力）。
  3. 风险因素（避免损失）。
  4. 自我形象（品牌与自我一致）。

### 第三篇：个体因素与心理因素

#### 第六章：消费者的购买动机

* **P144 隐性动机**：消费者没有意识到或不愿承认的动机。
* **P147 趋避冲突**：想趋近目标又想避开而造成的冲突。
* **需要与动机的区别**
  * **动机**：内在心理过程或动力，行为的直接原因。
  * **需要**：有机体内部的不平衡状态，动机的源泉。
* **P142 双因素理论的启示**
  1. 保健因素（保证质量和基本要求）。
  2. 激励因素（提供附加价值、形象）。
  3. 创新是关键。
* **P146 动机冲突对营销的启示**
  1. **双趋冲突**：强化某选择的价值，减价/延期付款。
  2. **双避冲突**：宣传消除错误信念。
  3. **趋避冲突**：提供保证担保。

#### 第七章：消费者的知觉

* **P149 知觉**：人脑对刺激物整体反应，对感觉信息的加工解释。
* **P156 完形原则**：即使要素不全，也倾向发展成完整画面。
* **P163 认知质量**：对产品适用性和功能特性的主观理解。
* **P166 购买知觉风险**：因无法预料结果而产生的不确定性感觉。
* **P153 处理信息超载**
  1. 提供消费者需要的信息。
  2. 突出重要信息（详细），次要信息用表格/录像提供。
* **P160 造成误解的原因与避免**
  * **原因**：刺激物模糊、注意力不集中、知识局限、误导性信息。
  * **避免**：预先测试信息。
* **P166 知觉风险类型**
  * 功能、物质、经济、社会、心理风险。
  * **原因**：新产品无体验、以往有不满意经历、机会成本、缺乏信心、产品复杂。

#### 第九章：态度的形成与改变

* **P203 态度**：对事物的评价、情感感受和行为倾向。
* **P223 霍夫兰德说服模式**
  * 涉及四个部分：外部刺激（传递者、传播、情境）、目标靶、中介过程（信息学习、感情迁移等）、劝说结果（改变或抵制）。
* **抵制劝说的方法**：贬损信源、歪曲信息、掩盖拒绝。
* **P224 名人信息源**
  * 特征：权威性、可靠性、外表吸引力、受众喜爱程度。

#### 第十章：个性、自我概念与生活方式

* **P243 品牌个性**：给拥有者带来溢价的无形资产，载体是名称/标志，源泉是消费者心智中的印象。
* **P243 自我概念类型**
  1. 实际的自我概念。
  2. 理想的自我概念。
  3. 社会的自我概念。
  4. 理想的社会自我概念。
  5. 期待的自我。
* **P244 象征品特征**：使用可见性、变动性、拟人化性质。
* **P248 AIO问卷**：活动 (Activity)、兴趣 (Interest)、意见 (Opinion)。
